# Корреляция

● Контекст
Корреляция - статистическая мера, используемая для оценки связи между двумя случайными величинами.

● Ваша задача
Написать скрипт для расчета корреляции Пирсона между двумя случайными величинами (двумя массивами). 
Можете использовать любую парадигму, но рекомендую использовать функциональную, т.к. в этом примере она значительно упростит вам жизнь.


## Function 'corrcoef' from module 'numpy'

Функция возвращает коэффициенты корреляции моментов произведения Пирсона

### numpy.corrcoef

numpy.corrcoef(x , y=None , rowvar=True , *, dtype=None ) 

Возвращение коэффициентов корреляции моментов произведения Пирсона.

Связь между матрицей коэффициентов корреляции R и ковариационной матрицей C равна

\[R_{ij} = \frac{ C_{ij} } { \sqrt{ C_{ii} C_{jj} } }\]

Значения R находятся в диапазоне от -1 до 1 включительно.

### Параметры функции 

#### xarray_like

Одномерный или двумерный массив, содержащий несколько переменных и наблюдений. Каждая строка x представляет переменную, а каждый столбец — отдельное наблюдение всех этих переменных. Также см. rowvar ниже.

#### yarray_like, optional

Дополнительный набор переменных и наблюдений. y имеет ту же форму, что и x .

#### rowvarbool, optional

Если rowvar — True (default) , то каждая строка представляет переменную с наблюдениями в столбцах. В противном случае связь транспонируется: каждый столбец представляет переменную, а строки содержат наблюдения.

#### dtypedata-type, optional

Тип данных результата. По умолчанию тип возвращаемых данных будет иметь точность не ниже numpy.float64.

### Function 'corrcoef' возвращает матрицу

Rndarray

Матрица коэффициентов корреляции переменных.
